hn2instapaper
=============

Save the top HN articles to Instapaper.

# Usage

```
hn2instapaper.

Save top HN articles to Instapaper.

Usage:
hn2instapaper <username> <password>
hn2instapaper -h | --help
hn2instapaper --version

Options:
-h --help     Show this screen.
--version     Show version.
```
